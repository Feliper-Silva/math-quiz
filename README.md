# math-quiz

# Sobre o projeto
Um quiz de perguntas e respostas sobre equações matematicas(mutiplicações), que mostra quando o usuário acertou e errou a pergunta. Além de contar os pontos que o usuário conseguiu, cada acerto=1 ponto.



## tecnologias usadas:

### front-end
 - HTML5
 - CSS
 - JavaScript + JQuery
